Additional Information: Automatic Image Path

When Automatic is selected whilst adding a product, its assuming that you are following existing file management patterns:

How the path is calculated:

"../resources/"  + categoryName + "/" + productName + "/" + productVariant + ".png";

/* example input */
product = {
name: "KARA"
category: "Keyboards"
variant: "AZUR"
}

/* example output*/
picturePath = "../resources/Keyboards/KARA/AZUR"

In case you are entering the file path manually, please make sure the directory you are entering is relative to ./site

Additional Information: How Products are displayed

All products that have the same name and category are categorized into ProductLines , for example, we have multiple products called KARA under category id 1, but have different descriptions, variants(color) and price. These products will be classified and displayed in their respective pages.
